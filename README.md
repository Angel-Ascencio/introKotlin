#Introduccion a kotlin

Practicas de kotlin elaboradas en clase para aprender el lenguaje.
Las practicas incluyen:
* Dos menus.
* Apliciación de cine para costo de boletos y descuentos.
* Diccionario de palabras en un idioma a otro.
* Saludar.
* Ejemplo 1.- Suma, resta, multipliación y divición de dos digitos.
* Ejemplo 5.- Un tipo de blog de notas.
* Calcular area de diferentes preguntas.
* Temas basicos.
